# Hangman

## Developers

For easy development, with live page reload, the globally installed "lite-server" npm package is used.
```bash
npm i -g lite-server sass
```
or
```bash
yarn add -g lite-server sass
```

## Commands terminal

### Development

```bash
lite-server
```
### Building Styles from Sources

```bash
npm run build:style
```
